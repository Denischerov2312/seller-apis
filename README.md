# Репизоторий для работы с данным ozon, Яндекс маркет.

## seller.py
Работает с маркетплэйсом OZON.
### Что умеет?
Скрипт имеет несколько фунцкий:
* Cкачивает данные об остатках товара, с сайта casio.
* Отправляет запрос об изменении количества товаров на сайт OZON.
* Изменяет цены товара в базе данных OZON.
### Что потребуется для работы:
Для доступа к изменения данных, вы должно иметь токен продавца, получаемый на сайте OZON.
А также надо иметь id клиента. Эти данные записываются в `.env` файл, расплагающийся в корне проекта.
Данные записываются в формате:
`ПЕРЕМЕННАЯ`=`ЗНАЧЕНИЕ`
Пример:
```
SELLER_TOKEN=13lkjlkf
CLIENT_ID=2131jkljldf
```
### Как запустить?
Для запуска необходимо иметь python третьей версии.
Необходимо установить репозиторий себе на ПК.
И запустить его командой:
```
python seller.py
```
